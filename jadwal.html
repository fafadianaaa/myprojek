Jadwal Latihan Mingguan

    Senin: Chest Day

    1. Bench Press

    Fokus Otot: Dada.

    Set & Repetisi: 4 set dengan 8-12 repetisi.

    Posisi Tubuh: Berbaring rata di bangku dengan punggung sedikit melengkung. Kaki menapak di lantai untuk keseimbangan.

    Gerakan: Turunkan bar perlahan ke dada dengan siku membentuk sudut 45 derajat dari tubuh. Dorong bar ke atas tanpa meluruskan siku sepenuhnya.

    Pernapasan: Tarik napas saat menurunkan bar, hembuskan saat mendorong ke atas.

    2. Incline Dumbbell Press

    Fokus Otot: Bagian atas dada.

    Set & Repetisi: 4 set dengan 8-12 repetisi.

    Posisi Tubuh: Duduk di bangku dengan kemiringan 30-45 derajat, dumbbell sejajar dada.

    Gerakan: Dorong dumbbell ke atas hingga lengan hampir lurus, turunkan perlahan ke posisi awal.

    Pernapasan: Tarik napas saat menurunkan dumbbell, hembuskan saat mendorong ke atas.

    3. Chest Fly

    Fokus Otot: Dada bagian dalam.

    Set & Repetisi: 3 set dengan 10-12 repetisi.

    Posisi Tubuh: Berbaring di bangku datar dengan dumbbell di kedua tangan, telapak tangan saling berhadapan.

    Gerakan: Rentangkan tangan perlahan ke samping hingga terasa regangan di dada, kembali ke posisi awal dengan kontrol.

    Pernapasan: Tarik napas saat merentangkan, hembuskan saat mengangkat.

    4. Push-Up

    Fokus Otot: Dada dan tricep.

    Set & Repetisi: 3 set dengan 15-20 repetisi.

    Posisi Tubuh: Tubuh lurus dari kepala hingga tumit, tangan sedikit lebih lebar dari bahu.

    Gerakan: Turunkan tubuh hingga dada hampir menyentuh lantai, dorong kembali ke posisi awal.

    Pernapasan: Tarik napas saat turun, hembuskan saat naik.

    Selasa: Bicep Day

    1. Barbell Curl

    Fokus Otot: Bicep.

    Set & Repetisi: 4 set dengan 8-12 repetisi.

    Posisi Tubuh: Berdiri tegak, kaki selebar bahu, pegangan barbell dengan telapak tangan menghadap ke atas.

    Gerakan: Angkat barbell ke arah dada dengan siku tetap di dekat tubuh, turunkan perlahan ke posisi awal.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.

    2. Dumbbell Hammer Curl

    Fokus Otot: Bicep brachialis.

    Set & Repetisi: 3 set dengan 10-12 repetisi.

    Posisi Tubuh: Berdiri tegak dengan dumbbell di kedua tangan, telapak tangan saling berhadapan.

    Gerakan: Angkat dumbbell dengan telapak tangan saling berhadapan, turunkan perlahan ke posisi awal.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.

    3. Concentration Curl

    Fokus Otot: Isolasi bicep.

    Set & Repetisi: 3 set dengan 10-12 repetisi.

    Posisi Tubuh: Duduk dengan siku bertumpu di paha bagian dalam, pegangan dumbbell dengan telapak tangan menghadap ke atas.

    Gerakan: Angkat dumbbell perlahan ke arah bahu, turunkan perlahan hingga siku hampir lurus.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.

    4. Preacher Curl

    Fokus Otot: Puncak kontraksi bicep.

    Set & Repetisi: 3 set dengan 8-12 repetisi.

    Posisi Tubuh: Duduk di bangku preacher dengan lengan bertumpu pada pad, pegangan barbell dengan telapak tangan menghadap ke atas.

    Gerakan: Angkat barbell ke arah bahu dengan kontrol, turunkan perlahan hingga siku hampir lurus.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.

    Rabu: Back Day

    1. Deadlift

    Fokus Otot: Punggung bawah, glutes, hamstring.

    Set & Repetisi: 4 set dengan 5-8 repetisi.

    Posisi Tubuh: Berdiri dengan kaki selebar pinggul, barbell di depan Anda.

    Gerakan: Angkat barbell dengan menjaga punggung tetap lurus, turunkan dengan kontrol.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.

    2. Bent-Over Row

    Fokus Otot: Punggung tengah.

    Set & Repetisi: 4 set dengan 8-12 repetisi.

    Posisi Tubuh: Membungkuk di pinggul dengan punggung lurus, pegangan barbell lebih lebar dari bahu.

    Gerakan: Tarik barbell ke arah perut, turunkan perlahan.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat menarik.

    3. Lat Pulldown

    Fokus Otot: Punggung atas (latissimus dorsi).

    Set & Repetisi: 4 set dengan 8-12 repetisi.

    Posisi Tubuh: Duduk tegak dengan pegangan lebar pada tali lat pulldown.

    Gerakan: Tarik tali ke bawah hingga hampir menyentuh dada, turunkan perlahan.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat menarik.

    4. Dumbbell Row

    Fokus Otot: Punggung tengah dan teratas.

    Set & Repetisi: 3 set dengan 8-12 repetisi.

    Posisi Tubuh: Satu lutut dan satu tangan di bangku, tangan lainnya memegang dumbbell.

    Gerakan: Angkat dumbbell ke arah pinggang, turunkan perlahan.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat menarik.

    Kamis: Tricep Day

    1. Tricep Dips

    Fokus Otot: Tricep.

    Set & Repetisi: 4 set dengan 8-12 repetisi.

    Posisi Tubuh: Gunakan dip bar, tubuh tegak dan tangan di pegangan bar.

    Gerakan: Turunkan tubuh hingga siku membentuk sudut 90 derajat, dorong tubuh ke atas kembali ke posisi awal.

    Pernapasan: Tarik napas saat menurunkan tubuh, hembuskan saat mendorong ke atas.

    2. Tricep Pushdown

    Fokus Otot: Tricep.

    Set & Repetisi: 4 set dengan 10-12 repetisi.

    Posisi Tubuh: Berdiri tegak dengan pegangan tali pada mesin kabel, tangan di atas kepala.

    Gerakan: Dorong tali ke bawah hingga tangan hampir lurus, kontrol saat kembali ke posisi awal.

    Pernapasan: Tarik napas saat kembali, hembuskan saat menurunkan.

    3. Close-Grip Bench Press

    Fokus Otot: Tricep dan dada bagian dalam.

    Set & Repetisi: 3 set dengan 8-12 repetisi.

    Posisi Tubuh: Berbaring di bangku dengan tangan lebih sempit dari lebar bahu.

    Gerakan: Turunkan bar ke dada bagian bawah, dorong ke atas kembali ke posisi awal.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mendorong.

    4. Skull Crushers

    Fokus Otot: Tricep.

    Set & Repetisi: 3 set dengan 8-12 repetisi.

    Posisi Tubuh: Berbaring di bangku dengan barbell atau dumbbell, tangan lurus ke atas.

    Gerakan: Turunkan bar ke dahi dengan siku tetap diam, angkat kembali ke posisi awal.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.

    Jumat: Shoulder Day

    1. Overhead Barbell Press

    Fokus Otot: Bahu bagian depan dan tengah.

    Set & Repetisi: 4 set dengan 8-10 repetisi.

    Posisi Tubuh: Berdiri tegak, pegangan barbell sedikit lebih lebar dari bahu.

    Gerakan: Angkat barbell di atas kepala, kontrol saat menurunkan kembali ke dada.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.

    2. Dumbbell Lateral Raise

    Fokus Otot: Bahu bagian tengah (deltoid).

    Set & Repetisi: 4 set dengan 10-12 repetisi.

    Posisi Tubuh: Berdiri tegak dengan dumbbell di kedua tangan, sedikit tekuk siku.

    Gerakan: Angkat dumbbell ke samping hingga sejajar bahu, turunkan perlahan ke posisi awal.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.

    3. Front Raise

    Fokus Otot: Bahu bagian depan.

    Set & Repetisi: 3 set dengan 10-12 repetisi.

    Posisi Tubuh: Berdiri tegak dengan dumbbell di kedua tangan di depan paha.

    Gerakan: Angkat dumbbell ke depan hingga sejajar dengan bahu, turunkan perlahan.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.

    4. Arnold Press

    Fokus Otot: Bahu depan dan tengah.

    Set & Repetisi: 3 set dengan 8-10 repetisi.

    Posisi Tubuh: Duduk atau berdiri, pegang dumbbell dengan telapak tangan menghadap wajah.

    Gerakan: Putar tangan dan angkat dumbbell ke atas kepala, turunkan perlahan.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.

    Sabtu: Leg Day

    1. Squat

    Fokus Otot: Paha depan, glutes, hamstring.

    Set & Repetisi: 4 set dengan 8-12 repetisi.

    Posisi Tubuh: Berdiri dengan kaki selebar pinggul, barbell di punggung atas.

    Gerakan: Turunkan tubuh hingga paha sejajar dengan lantai, dorong kembali ke atas.

    Pernapasan: Tarik napas saat turun, hembuskan saat naik.

    2. Leg Press

    Fokus Otot: Paha depan, glutes.

    Set & Repetisi: 4 set dengan 10-12 repetisi.

    Posisi Tubuh: Duduk di mesin leg press, kaki menekan platform dengan lebar selebar bahu.

    Gerakan: Dorong platform ke atas hingga kaki hampir lurus, turunkan perlahan.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mendorong.

    3. Lunges

    Fokus Otot: Paha depan, glutes, hamstring.

    Set & Repetisi: 3 set dengan 10-12 repetisi per kaki.

    Posisi Tubuh: Berdiri tegak dengan dumbbell di tangan, langkah maju dengan satu kaki.

    Gerakan: Turunkan tubuh hingga lutut membentuk sudut 90 derajat, dorong kembali ke posisi awal.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat dorong.

    4. Calf Raises

    Fokus Otot: Betis.

    Set & Repetisi: 4 set dengan 15-20 repetisi.

    Posisi Tubuh: Berdiri dengan kaki sedikit terpisah, angkat tumit setinggi mungkin.

    Gerakan: Angkat tubuh menggunakan betis, turunkan perlahan.

    Pernapasan: Tarik napas saat menurunkan, hembuskan saat mengangkat.
